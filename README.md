## 简要说明：
#### 最近面试比较多，准备面试的比较烦，做了一个skill，用于面试前的准备，包括公司信息调研、岗位匹配度分析、岗位面试问题预测和回答、定制化个人介绍等。

  
## 使用方式：
1、打开你的openclaw（workbuddy/kimiclaw/minclaw等），把md文件和html文件下载完拖进去。  
2、准备**你的简历**（txt、word和PDF都行，txt/word最好解析更快），和你要准备的**岗位描述**，以boss直聘为例，在手机APP上点击分享-复制链接-电脑打开-复制完整JD（岗位描述）。  
3、然后把你**个人简历**、**公司名称**、**完整岗位描述**(JD)填进去，直接调用skill让他给你准备这个岗位的面试。  
4、等待输出结果，结果会以html格式输出，直接**用浏览器打开**即可。
